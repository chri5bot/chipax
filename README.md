## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```

## Test

```bash
# unit tests
$ npm run test

```

## Para probar los ejercicios tienes que realizar un get desde postman

```bash
$ localhost:5001/rick-and-morty/char-counter-exercise
```

```bash
$ localhost:5001/rick-and-morty/episode-locations
```
